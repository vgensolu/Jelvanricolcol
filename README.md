# Jelvan Ricolcol - Portfolio

Professional HR portfolio website built with HTML, Tailwind CSS, and JavaScript.

## Live Site

Visit the portfolio at: `https://vgensolu.github.io/Jelvanricolcol/`

## Local Development

To view the site locally, open `index.html` in your web browser or use a local web server:

```bash
# Using Python 3
python3 -m http.server 8000

# Using Node.js
npx http-server -p 8000
```

Then navigate to `http://localhost:8000`
